# KaNote - 舰C笔记本

一个为POI设计的舰队Collection笔记本插件，帮助你记录建造数据、游戏心得和舰队管理信息。

## 功能特色

### 📝 笔记本功能
- **多分类笔记**: 支持通用、建造记录、远征记录、战斗记录、攻略笔记、舰娘图鉴等分类
- **实时保存**: 自动保存功能，永不丢失重要记录
- **搜索功能**: 快速搜索笔记标题、内容和标签
- **标签系统**: 为笔记添加标签，便于分类管理
- **模板功能**: 内置建造、远征、战斗记录模板

### 📊 数据管理
- **建造记录**: 记录建造资源消耗和结果
- **远征记录**: 跟踪远征成功率和收益
- **统计分析**: 自动生成建造成功率、远征统计等数据
- **舰队信息**: 管理舰娘数据和等级信息

### 💾 数据安全
- **自动备份**: 定期自动备份重要数据
- **手动备份**: 支持手动创建和恢复备份
- **数据导出**: 支持导出笔记为文本格式
- **配置管理**: 个性化设置和主题配置

## 安装说明

### 快速安装（推荐）

1. **下载并解压插件包**
2. **双击运行 `build.bat` 构建插件**
3. **双击运行 `install-to-poi.bat` 自动安装**
4. **重启POI并在插件设置中启用KaNote**

### 手动安装

1. **构建插件**
   ```bash
   npm install
   npm run build
   ```

2. **找到POI插件目录**
   - Windows: `POI安装目录\resources\app\plugins`
   - Steam版: `C:\Program Files (x86)\Steam\steamapps\common\poi\resources\app\plugins`

3. **复制插件**
   - 将整个 `poi-plugin-kanote` 文件夹复制到plugins目录

4. **启用插件**
   - 重启POI
   - 设置 → 插件 → 启用KaNote

### 使用方法
- 在POI主界面点击"舰C笔记本"标签
- 或通过插件菜单访问

## 使用方法

### 创建笔记本
1. 点击侧边栏的"+"按钮或主页面的"创建新笔记本"按钮
2. 选择合适的分类
3. 输入标题和内容
4. 添加标签（可选）
5. 点击保存

### 使用模板
在编辑器工具栏中选择对应的模板：
- **建造模板**: 快速记录建造资源消耗和结果
- **远征模板**: 记录远征配置和收益
- **战斗模板**: 记录出击海域和战斗结果

### 数据管理
1. 切换到"舰队数据"标签
2. 在对应的记录页面添加数据
3. 查看统计分析结果

### 搜索笔记
在侧边栏搜索框中输入关键词，支持搜索：
- 笔记标题
- 笔记内容
- 标签

### 技术栈
- **前端框架**: React 18
- **样式**: CSS3
- **构建工具**: Webpack 5
- **存储**: 本地文件系统（fs-extra）
- **POI集成**: Electron APIs

## 配置选项

插件支持以下配置选项：
- 主题设置
- 自动保存间隔
- 自动备份设置
- 字体和界面设置
- 快捷键配置

## 数据存储

所有数据存储在用户数据目录的`kanote`文件夹中：
- `notebooks/`: 笔记本文件
- `kancolle-data/`: 舰C游戏数据
- `backups/`: 备份文件
- `config.json`: 配置文件
